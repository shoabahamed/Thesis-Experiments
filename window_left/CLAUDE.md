# Environment

All Python/ML commands (training, testing, scripts, notebooks, etc.) in this project must be run using the following Anaconda environment:

```
C:\Shoab\ML\environments\anaconda-env\pytorch_gpu
```

Activate it before running any command, e.g.:

```
conda activate C:\Shoab\ML\environments\anaconda-env\pytorch_gpu
```

Do not use the base environment or any other environment for running code in this repository.
